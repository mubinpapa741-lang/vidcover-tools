# VidCover Tools

Premium subscription-based video cover & thumbnail tools app (PWA).

## Features
- 3 subscription plans (Starter, Pro, Ultimate)
- Code-based plan activation (30 days)
- Admin dashboard (password protected)
- Installable as mobile app (PWA)
- Auto-update support
- Works on Android, iOS, PC, Mac

## Quick Start
```bash
pip install -r requirements.txt
python app.py
```

## Deployment
This app is ready for deployment on Render.com (free tier).
